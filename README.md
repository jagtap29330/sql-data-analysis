# SQL Data Analysis Projects

This repository contains SQL-based data analysis projects demonstrating real-world business analytics skills.

## Skills Demonstrated
- SQL Joins
- Common Table Expressions (CTEs)
- Window Functions
- Aggregations & KPI Analysis

## Projects

### Sales Analysis
**File:** `sales_analysis.sql`

Analyzes sales performance by product category using joins, CTEs, and ranking functions to support data-driven decisions.

**Business Impact:**
- Identifies top-performing product categories
- Supports data-driven revenue decisions


### Customer KPI Analysis
**File:** `customer_kpi.sql`

Calculates customer-level KPIs such as total orders, total spending, and customer ranking.

## Tools
- SQL (MySQL / PostgreSQL / SQL Server compatible)
